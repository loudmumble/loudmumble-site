# loudmumble-site

Personal portfolio site for loudmumble.com. Built with Vite + React + TypeScript + Tailwind CSS.

## Prerequisites

- Node.js 18+
- npm 9+

## Install

```bash
npm install
```

## Development

```bash
npm run dev
# → http://localhost:5173
```

## Build

```bash
npm run build
# → dist/  (static files, deploy anywhere)
```

Preview the production build locally:

```bash
npm run preview
```

## Lint

```bash
npm run lint
```

## Test

```bash
npm test
```

## Deploy

The `dist/` directory is a static site — deploy to Vercel, Netlify, Caddy, nginx, or any static host. The `CNAME` file targets `loudmumble.com`.
